## Electron app.
### Basic setup with React, Typescript and Webpack.

To run this project.

- Clone this repo

```bash
  git clone https://github.com/elisealcala/electron-react-ts.git
```

- Install the dependencies

```bash
  npm install
```

- run this commands in  console.


```bash
  npm run dev:electron
